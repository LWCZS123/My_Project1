# My_Project1 个人记账App

## 项目简介
本项目是一款轻量化Android记账应用，面向个人日常收支管理，操作简洁，数据本地云端双存储，支持消费统计图表展示。

## 应用截图
![首页](images/main.jpg)
![记账页面](images/add_record.jpg)
![统计图表](images/chart.jpg)

## 主要功能
1. 收支快速录入，支持备注、金额、分类
2. 按月/按日筛选消费记录
3. 饼图展示各类支出占比
4. 数据云端同步备份

## 开发环境
- 编译SDK：Android 14
- 开发工具：Android Studio
- 语言：Java
- 第三方框架：Bmob、Glide、MPAndroidChart

## 部署步骤
1. Clone本仓库到本地
2. 使用Android Studio打开项目
3. 在build.gradle配置自己的Bmob Application ID
4. 连接安卓模拟器/真机运行
